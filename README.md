# 🌿 Organic Product Website Clone

A fully responsive website clone, created using **HTML** and **CSS**. The project focuses on clean layout structuring and modern UI interactions using CSS Flexbox.

## 🛠️ Built With

- **HTML5** – Semantic markup for structure
- **CSS3** – Styling, Flexbox layout, transitions, and hover animations

## 📂 Project Structure

organic-product-website/ <br>
├── index.html <br>
├── style.css <br>
├── /images <br>
├── Orbito.otf <br> 
└── README.md

## 🎯 What I Learned / Practiced

- Structuring a multi-section webpage with semantic HTML.
- Creating responsive layouts using CSS Flexbox (no frameworks).
- Implementing hover effects and simple CSS animations/transitions for interactive navigation.
- Designing a cohesive visual theme suited to an organic/eco-friendly brand.
